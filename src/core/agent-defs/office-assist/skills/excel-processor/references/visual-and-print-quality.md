# Visual and Print Quality

Design for the workbook's function. A dense operational sheet, a fixed printed form, and an executive summary should not share one visual recipe.

## Hierarchy and layout

- Establish a clear title, context/period, headers, data area, totals, and notes where relevant.
- Use alignment, whitespace, borders, and type weight before adding color.
- Keep recurring fields and controls in predictable positions.
- Set useful column widths and row heights; wrap long labels deliberately.
- Freeze the region users need while scrolling.

## Color and typography

- Use one neutral system plus a small number of semantic accents.
- Reserve strong color for actions, exceptions, or the most important result.
- Distinguish editable, calculated, reference, and output cells subtly and consistently.
- Never rely on color alone; retain labels, symbols, or status text.
- Avoid rainbow headers, excessive fills, 3D effects, and ornamental gradients.

## Numbers and tables

- Match number formats to meaning: count, amount, percentage, date, duration, or identifier.
- Put units in headers or adjacent labels instead of mixing them into numeric values.
- Align comparable numbers and use consistent precision.
- Use banding sparingly; emphasize totals and exceptions rather than every boundary.
- Prefer tables for records and fixed cells for forms; do not force one structure into the other.

## Charts and summaries

- Choose chart type by question: line for trend, bar for comparison, scatter for relationship, histogram for distribution.
- Avoid 3D charts, redundant legends, heavy borders, and unnecessary gridlines.
- Highlight the series or point that supports the intended observation and mute context.
- Add direct labels or annotations when they reduce lookup effort.
- Do not make a dashboard when a small table answers the question better.

## Print quality

- Set paper size, orientation, margins, scaling, print area, repeating rows/columns, and page order explicitly.
- Keep headers, totals, signatures, and notes together where possible.
- Add page numbers, document identifiers, version/date, and confidentiality markings when appropriate.
- Inspect PDF/print preview for clipped text, isolated headings, blank pages, and unreadable scaling.
