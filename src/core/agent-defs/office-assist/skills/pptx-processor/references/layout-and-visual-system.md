# Layout and Visual System

Create a coherent system, then adapt composition to each slide's job.

## Foundation

- Use the source template's dimensions; otherwise default to 16:9 for modern screen delivery.
- Define theme fonts that cover the presentation language, including Japanese glyphs when needed.
- Define a neutral palette, one primary color, and a restrained accent/semantic palette.
- Establish consistent outer margins, columns, gaps, title position, footer behavior, and alignment anchors.
- Reuse components for titles, page numbers, source notes, callouts, cards, and section markers.

## Hierarchy and typography

- Make title, key message, evidence, annotation, and source visibly distinct.
- Size text for the delivery context; projection requires larger type than a circulated document.
- Limit font families and weights. Avoid using many type sizes to compensate for weak structure.
- Keep line lengths readable and use spacing before shrinking text.
- Handle Japanese line breaking, punctuation, and mixed-script alignment deliberately.

## Composition patterns

- Use title plus focal statement for a key message.
- Use two columns for comparison, before/after, or text plus evidence.
- Use a grid of cards only for genuinely parallel items.
- Use a process, timeline, or flow layout for sequence and dependency.
- Use a chart-led layout when data is the evidence.
- Use a full-bleed image only when the image carries meaning and remains legible with overlays.
- Use appendix/reference layouts for dense tables and definitions instead of forcing them into narrative slides.

## Restraint and consistency

- Align objects precisely and keep equal gaps visibly equal.
- Prefer whitespace and contrast to boxes around every element.
- Avoid 3D effects, gratuitous gradients, excessive shadows, random icons, and mixed image treatments.
- Use decoration only when it reinforces hierarchy, grouping, sequence, or brand.
- Vary layout to serve content while retaining the same visual grammar.

## Images and diagrams

- Use relevant, high-resolution, legally usable images; record sources when required.
- Crop consistently and avoid distortion.
- Keep icon style, stroke weight, and perspective consistent.
- Build diagrams with clear reading order, labeled relationships, and minimal crossing lines.
- Do not use stock photos as filler for abstract concepts when a direct diagram or evidence would communicate better.
